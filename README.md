# Exposure

Smart contract generation and connect-to-escrow infrastructure for gig performers.

## Tech stack

- PHP 8.3+ and Laravel 13
- MySQL 8
- Tailwind CSS v4 via Vite
- DaisyUI (CSS components for Tailwind) for the user-facing components
- Livewire Volt for inline interactive components
- Laravel Breeze for authentication
- Filament v5 admin panel
- Pulse for application performance insights at /pulse
- Pest v4 for tests
